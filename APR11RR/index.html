Behind the numbers
How the simulation works

These projections come from a Monte Carlo simulation built entirely on our team's actual game logs and head-to-head results — not generic stats or guesswork. Here's what powers it:

10,000 simulated games per matchup. Each cycle plays out all 6 innings pitch-by-pitch, generating a statistical distribution of outcomes.

Real game logs only. 27 Gambrill games, 46 HCYP games, and 35 BCC games were parsed to build each team's offensive and pitching profile.

Batting orders derived from play-by-play data — not assumed. Each player's lineup slot reflects their actual most-frequent position across the season.

Log5-style run expectancy combines each team's offensive strength with the opposing pitching staff's ERA. Pitcher-specific quality factors adjust each inning.

Same-day fatigue modeled. Each team's second game applies a 15% ERA penalty to reflect the real performance drop in back-to-back 6-inning outings.

Head-to-head history integrated. Where direct matchup data exists (we have 4 prior games against these opponents), it shapes pitcher recommendations.
